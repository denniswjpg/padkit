What is this?

PadKit is open firmware for the cheap 6-key + rotary-knob CH552 macropad. Flash it once and every key and the knob can send any shortcut, run a command, hit a webhook, or talk to an AI agent.

Mute & volume for calls · media + scrub · run tests / git on keys · Home Assistant scenes · approve & scroll an AI coding agent — you decide.

Flash in your browser

Chrome/Edge, nothing to install on macOS/Linux. The chip is unbrickable — a failed flash is always recoverable. First put the pad into bootloader mode:

  1. Remove the four faceplate screws, lift out the PCB.
  2. Bridge the SW2 pads (back, next to the U7 chip) with tweezers and hold while you plug in USB-C, then release.
  3. Hit Connect & flash below and pick the device — do it promptly, the bootloader window is only a few seconds.

CLI alternative: ./flasher/isp55e0 -f firmware/padkit.bin. Windows needs a one-time Zadig → WinUSB step. Details: docs/flashing.md.
